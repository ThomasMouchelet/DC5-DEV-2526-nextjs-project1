# Start project

```bash
  # Start containers
  docker compose up -d
  # OR
  docker-compose up -d

  # Install dependancies
  npm install

  # Start project
  npm run dev
```

```bash
  # update db schema
  npx prisma db push

  # Generated
  npx prisma generate

  # Seed
  npm run seed
```
